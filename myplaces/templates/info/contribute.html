{% trans 'How to contribute'%}

{%blocktrans%} MyPlaces is an open source project so anybody, who finds it compelling and interesting, can contribute. You can contribute by:
  • Financial donations
    See below for details how to make a donation to this project.
  • Developing the application
    Source code is on github so you can join us there. Required skills are python (django), Javascript, HTML5, GIS (PostGIS, Geodjango).
  • Documenting the software
    Are you enjoying creation of compelling user or developers documentation? You are more then welcomed - please join us on github.
  • Translation of the site
    Do you want to translate MyPlaces to other language? Go ahead and join us on github.
  • Proposing improvements and new features
    If you have any idea how to improve our service by new cool functionality, let us know.
{%endblocktrans%}
